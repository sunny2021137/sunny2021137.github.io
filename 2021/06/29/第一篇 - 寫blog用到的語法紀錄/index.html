第一篇 - 寫blog用到的語法紀錄

發表於 2021-06-29 更新於 2021-06-30 分類於 blog筆記 閱讀次數：

避免自己忘記語法，第一篇blog記錄寫blog常用到的語法。
剛學希望不要忘得太快啦！

hexo

在blog資料夾下執行：

1. 創建資料夾

2. 新建文章

3. 部署至github

4. 預覽

5. head部分

Markdown

1. 標題(1-6級)

2. 段落換行

2個以上空格加enter 或 空一行

3. 粗體、斜體(*、_)

4. 分隔線( * 、 - 、 _ )

5. 列表

  • 無序( *、 + 、 - )
  • 有序
  • 嵌套(子列表前加4空格)

6. 區塊

  • 基本
  • 嵌套

7. 代碼區

用、、、前後包裹

8. link

  • 文字顯示連結
  • 直接顯示地址連結

9. 圖片

使用img標籤可設寬高！

10. 表格

  • 基本
  • 對齊方式
    • -:：右對齊
    • :-：左對齊
    • :-:： 居中對齊
